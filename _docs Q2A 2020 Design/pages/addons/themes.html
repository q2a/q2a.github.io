Question2Answer Themes

To install a theme, place its directory in Q2Aâ€™s qa-theme directory, then open the General section of the Admin panel and choose the theme from the menu provided. Many good themes can be found at Q2A service providers. Below is a list of other recently updated themes:

Free themes

  • Mayro (A Google Material Design Theme) by Momin Raza.
  • Brown-Clean by Abhinav.
  • Default Right-To-Left (based on Q2A Default theme) by Towhid Nategheian @ AskWeb.
  • Donut (responsive) by Amiya Sahu.
  • Ejemplo de by Mauricio @ Ejemplo de.
  • Esteem Theme (responsive) by Towhid @ QA-Themes.
  • Google-style Mobile Theme by NoahY.
  • IdeaBox Theme (responsive) by Towhid @ QA-Themes.
  • Metro Theme by Towhid @ QA-Themes.
  • Minimalist Answer by Digitizor Media.
  • Mobile Theme by NoahY. Before Q2A 1.5, use the Theme Switcher Plugin to automatically switch themes for mobile users.
  • Parchment Theme by NoahY.
  • Sharp by Towhid @ QA-Themes.
  • Studio by Towhid @ QA-Themes.
  • Sturtz Mobile by Nate Sturtz @ Sturtz Network. Mobile, responsive, designed for SEO.
  • Stylish (responsive) by Towhid @ QA-Themes.
  • Support Theme by Towhid @ QA-Themes.
  • Twenty Eleven by Maximilian Narr. For use with the WordPress theme of the same name.
  • TwentyTwelve by QA-Themes. Based on WordPress default theme, TwentyTwelve.
  • Twitter Bootstrap by Harsh J. Vakharia. For Q2A 1.5.3 only, since it heavily modifies the core to work with Bootstrap.

Paid themes

  • AVEN (Material Design Theme) by Chun.
  • Muffin (Material Design Theme) by Chun.
  • Frapuchino (Material Design Theme) by Chun.
